## 📚 Syllabus & Progress Tracker

Track my learning journey through the major modules of the course. *(Checked boxes `[x]` indicate completed modules)*

### 1. Fundamentals & Basics
- [x] **Introduction:** Flowcharts, pseudocode, and environment setup.
- [ ] **Language Basics:** Variables, data types, operators, conditional statements, loops, and pattern printing.
- [ ] **Functions:** Implementation, scope, and function execution mechanics.

### 2. Number Systems & Bit Manipulation
- [ ] **Binary Number System:** Conversions between decimal and binary.
- [ ] **Bitwise Operators:** Bitwise AND, OR, XOR, shifts, and data type modifiers.

### 3. Arrays, Vectors, & Hashing
- [ ] **1D Arrays & Vectors:** Introduction, dynamic sizing with vectors, and basic operations.
- [ ] **Advanced Array Algorithms:** Kadane's Algorithm, Moore's Voting Algorithm, Two-Pointer approach, and Prefix/Suffix sums.
- [ ] **2D Arrays:** Matrix operations, searching in a sorted 2D matrix, and Spiral Matrix traversal.
- [ ] **Basic Hashing Problems:** Two Sum, 3 Sum, 4 Sum, and Subarray Sum Equals K.

### 4. Pointers & Strings
- [ ] **Pointers:** Core concepts, memory addresses, and pointer arithmetic.
- [ ] **Strings & Character Arrays:** Palindrome checking, string permutations, reverse words, and string compression.

### 5. Searching & Sorting Algorithms
- [ ] **Binary Search:** Iterative/recursive approaches, and advanced problems (Search in Rotated Sorted Array, Book Allocation, Painter's Partition, Aggressive Cows).
- [ ] **Sorting:** Bubble Sort, Selection Sort, Insertion Sort, DNF (Dutch National Flag) Sort, Merge Sort, and Quick Sort.

### 6. Core Mathematics for DSA
- [ ] **Math Concepts:** Euclid's Algorithm for GCD, Sieve of Eratosthenes for prime numbers, and Modular Arithmetic.

### 7. Recursion & Backtracking
- [ ] **Recursion Basics:** Standard problems like Fibonacci sequences and array verification.
- [ ] **Backtracking:** Advanced puzzles (Subsets generation, Permutations, N-Queens, Sudoku Solver, Rat in a Maze).

### 8. Object-Oriented Programming (OOPs)
- [ ] **OOPs Concepts:** Classes, objects, inheritance, polymorphism, encapsulation, and abstraction for technical interviews.

### 9. Advanced Data Structures
- [ ] **Linked Lists:** Singly, Doubly, and Circular lists; reversing lists, cycle detection/removal, and flattening.
- [ ] **Stacks:** Implementation, Valid Parentheses, Stock Span, Next Greater/Smaller Element, and Histogram problems.
- [ ] **Queues:** Circular Queues, Stacks using Queues (and vice versa), Sliding Window Maximum, and LRU Cache.
- [ ] **Binary Trees:** Traversals (Inorder, Preorder, Postorder, Morris), Height/Diameter, LCA, and View problems.
- [ ] **Binary Search Trees (BST):** Creation, validation, and balancing.
